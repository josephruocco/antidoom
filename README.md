# AntiDoom

Chrome extension prototype that injects fake popup ads with dry, positive anti-doomscroll messages.

## Load in Chrome

1. Open `chrome://extensions`.
2. Enable Developer mode.
3. Click Load unpacked.
4. Select `/Users/josephruocco/antidoom`.

## What it does

- Watches for sustained scrolling on any page.
- Shows a popup-style "sponsored" message after enough scrolling and time have passed.
- Lets you snooze the interruptions or tune frequency from the extension popup.
- Includes a bundled demo page with many example ads and a WordArt-style AntiDoom logo.

## Current tone

The copy aims for:

- positive without being cheesy
- slightly dry and confrontational
- more "your future self bought this ad" than generic wellness advice
